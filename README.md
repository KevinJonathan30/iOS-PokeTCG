## Features

- [x] Card List w/ Search Feature
- [x] View Card Detail
- [x] Recommendation Card
- [x] Click Card to Zoom (Detail Page)

## Requirements

- iOS 12.1+
- CocoaPods

## Installation

#### CocoaPods
Before running the project, please run 'pod install' command first.
